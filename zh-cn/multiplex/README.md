# 可复用资产建设

> 推荐使用 [dumi](https://d.umijs.org/zh-CN/guide/basic) 来进行文档建设

## 介绍

[dumi](https://d.umijs.org/zh-CN/guide/basic) 中文发音嘟米，是一款为组件开发场景而生的文档工具。在可复用资产建设的场景下有着非常大的优势。
## 快速开始


```
# one-cli 默认初始化组件库
one init component

# 初始化一个文档模式的组件库开发脚手架
npx @umijs/create-dumi-lib

# 初始化一个站点模式的组件库开发脚手架
npx @umijs/create-dumi-lib --site
```
